## Lab web mínimo (idea)

Monta o usa una app vulnerable **propia** (DVWA, Juice Shop, PortSwigger Academy).

Objetivo de 30–60 min:

1. Proxy con [Burp](/WEB-Notas-de-Hacking/web/burpbasico/).
2. Encuentra un reflejo → prueba [XSS](/WEB-Notas-de-Hacking/web/xss/).
3. Encuentra un param SQL → prueba [SQLi](/WEB-Notas-de-Hacking/web/sqli/).
4. Anota con la [plantilla](/WEB-Notas-de-Hacking/ctf/plantilla/).

No hace falta publicar el walkthrough completo aquí: el valor está en tu nota comando → resultado.
