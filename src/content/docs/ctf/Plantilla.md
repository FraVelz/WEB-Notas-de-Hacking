# Plantilla de walkthrough CTF

Copia esto por nivel/flag. Estilo [Bandit](/WEB-Notas-de-Hacking/linux/banditresolucion/).

```markdown
## Nivel N — título corto

**Objetivo:** (qué pide el enunciado)

**Pista / idea:** (una línea)

### Pasos

1. Comando
   `comando aquí`
2. Resultado relevante
   (pega solo lo útil)
3. Siguiente paso
   (por qué ese resultado abre la puerta)

**Flag / password:** `...`

**Qué aprendí:** (1 frase)
```

## Reglas

- No novelas: densidad > prosa.
- Si falló un intento, una línea “no funcionó X porque Y”.
- Enlaza notas del repo cuando el truco sea reutilizable (SQLi, SUID, etc.).
